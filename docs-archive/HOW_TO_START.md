# NetShark 使用指南

## 🚀 推荐启动方式

由于 PowerShell 后台模式存在参数传递问题，请使用**前台模式**（更稳定）：

```powershell
cd D:\PythonProject\NetShark
.\start.ps1
```

**特点**:
- ✅ 自动检测并安装 Npcap
- ✅ 启动两个窗口（显示后端和前端日志）
- ✅ 100% 稳定可靠

**窗口管理**:
- 最小化两个日志窗口即可
- 关闭窗口 = 停止服务（方便调试）

---

## 🛑 停止服务

关闭两个日志窗口即可，或运行：

```powershell
.\stop.ps1
```

---

## 📝 常见问题

### Q: 不想看到日志窗口？
A: 最小化两个窗口即可，它们会继续正常运行

### Q: 如何后台运行？
A: 最简单的方法：
1. 运行 `.\start.ps1`
2. 最小化两个窗口
3. 它们会一直运行直到您关闭
